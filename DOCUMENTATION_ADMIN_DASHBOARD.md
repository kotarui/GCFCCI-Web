## 1️⃣ What is the Admin Dashboard?

**Admin Dashboard** = Control center for managing all church content

Think of it like a restaurant manager's office:
- **Public Website** = Dining room (what customers see)
- **Admin Dashboard** = Manager's office (where you control everything)

The admin dashboard lets you:
- Add/edit/delete leaders, sermons, events, ministries
- View prayer requests and visitor registrations
- Change church settings (logo, address, phone, etc.)
- Manage announcements and news
- Upload photos, videos, and documents
- Edit the Ladders of Success stages

---

## 2️⃣ How to Access the Admin Dashboard

### Step 1: Go to Website
Visit: `https://your-church-website.com`

### Step 2: Click Admin Button
Look for the blue "Admin" button in the top-right corner

### Step 3: Log In
You'll be redirected to the login page. Log in with your credentials.

### Step 4: You're In!
You'll see the Admin Dashboard home page with all management options.

## 4️⃣ Leader Management

### What You Can Do:
- View all church leaders
- Add new leaders
- Edit existing leaders
- Delete leaders
- Reorder leaders (change display order)

### Adding a Leader:

**Step 1:** Click "Add Leader" button

**Step 2:** Fill out the form:
```
Name:                [Text field]
Position:            [Text field] (e.g., Senior Pastor)
Ministry:            [Text field] (e.g., Worship)
Biography:           [Large text area]
Photo URL:           [URL of leader's photo]
Favorite Bible Verse: [e.g., John 3:16]
Bible Verse Text:    [Full verse text]
Contact Email:       [Email address]
Contact Phone:       [Phone number]
```

**Step 3:** Click "Add Leader" button

**Step 4:** Leader appears in the list!

### Editing a Leader:

**Step 1:** Find the leader in the list

**Step 2:** Click the "Edit" button (pencil icon)

**Step 3:** Change the information

**Step 4:** Click "Update Leader"

### Deleting a Leader:

**Step 1:** Find the leader in the list

**Step 2:** Click the "Delete" button (trash icon)

**Step 3:** Confirm deletion

**Step 4:** Leader is removed!

## 1️⃣4️⃣ Ladders of Success Editor

### What It Is:
The Ladders of Success are the 4 stages of discipleship:
1. **WIN** - Winning souls (evangelism)
2. **CONSOLIDATE** - Consolidating believers (follow-up)
3. **DISCIPLE** - Discipling leaders (training)
4. **SEND** - Sending missionaries (deployment)

### What You Can Edit:
For each stage, you can edit:
```
Stage Name:           [WIN, CONSOLIDATE, DISCIPLE, SEND]
Title:                [Stage title]
Description:          [What this stage is about]
Bible Verse:          [Reference like John 3:16]
Bible Verse Text:     [Full verse text]
Icon URL:             [Stage icon image]
Image URL:            [Stage image]
```

### How to Edit:

**Step 1:** Click "Ladders of Success" in admin dashboard

**Step 2:** Click "Edit" on the stage you want to change

**Step 3:** Update the information

**Step 4:** Click "Update Stage"

**Step 5:** Changes appear on the Ladders page!

### Example Ladder Stage:
```
┌──────────────────────────┐
│ [Icon image]             │
│ WIN                      │
│ Winning Souls            │
│ Reaching people for...   │
│ John 3:16                │
│ [Edit]                   │
└──────────────────────────┘
```

## 🔒 Security & Permissions

### Who Can Access?
- Only **admins** can access the admin dashboard
- Regular visitors cannot see the admin area
- Login is required

### What Admins Can Do?
- Create, edit, delete all content
- View prayer requests
- View visitor registrations
- Change church settings
- Manage all pages

### What Visitors Cannot Do?
- Access admin dashboard
- Edit any content
- Delete anything
- See prayer requests
- Change settings

## 🆘 Troubleshooting

### Problem: Image not showing
**Solution:** Make sure the image URL is correct and the image is publicly accessible

### Problem: Can't add event
**Solution:** Make sure all required fields are filled in (title, date, time, venue)

### Problem: Changes not showing on website
**Solution:** Refresh the website page (press Ctrl+R or Cmd+R)

### Problem: Can't log in to admin
**Solution:** Make sure you're using the correct login credentials

### Problem: Forgot admin password
**Solution:** Contact the website administrator to reset your password
